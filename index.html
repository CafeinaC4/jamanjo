Gostaria de contribuir? Veja como fazer...

  • Com conteúdo

    Se você conhece algum recurso legal para aprendizes ou desenvolvedores e gostaria de compartilhar, você pode abrir um pull request no repositório aqui com o nome do recurso, uma breve descrição e seu link.

    Se abrir um pull request ainda lhe parece coisa de maluco, você pode enviar sua contribuição pelo e-mail rafaelld@gmail.com

  • Participando do desenvolvimento

    Além do propósito de divulgar coisas legais e ajudar a galera, este projeto também serve de laboratório. Se quiser contribuir implementando novas funcionalidades ou melhorando as gambiarras existentes, seja muito bem vindo com seu pull request!

  • Dando uma estrelinha no GitHub

    Dev biscoitero adora estrelinha no GitHub :P

    Se quiser massagear o ego do dev e contribuir com a relevância do projeto, clica aqui pra acessar o repositório e aperta o botão de estrelinha no canto superior direito!
